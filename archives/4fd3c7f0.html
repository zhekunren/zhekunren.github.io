4.以管理员的身份打开 cmd ，依次执行下面命令：

继续输入指令mysqld -install，会打印Service successfully installed；
继续输入指令 net start mysql，会打印提示MySQL服务已经启动成功。到这里，MYSQL5.6已经安装成功，并且已运行

5.MYSQL登陆与修改密码
① 在cmd中，输入 mysql -u root -p ，输入该指令后出现 Enter password: ，第一次登陆直接回车，进入mysql软件；
② 输入set password for root@localhost=password('root'); ，此时将root的密码设置为root；
③ 输入exit，退出mysql软件；
④ 再次输入mysql -u root -p 时，需要输入密码root后才能正常进入mysql软件。

6.为了方便后面QT5.5.1和Mysql的测试程序编写，新建一个mydatabase数据库，用于后面的程序操作。
① 在cmd中，输入 mysql -u root -p，登录成功后，输入create database mydatabase;；
② 输入show databases;，即可看到我们新建的mydatabase数据库。

7.问题补充：
经过测试上述安装后，数据库中的中文会显示成”???”乱码。经过分析，发现是前面D:\Software\mysql-5.6\bin\my-default.ini中的配置没有生效。解决步骤如下：
① win+R—->regedit—->HKEY_LOCAL_MACHINE—->SYSTEM—->CurrentControlSet—->Services—->MySQL，根据路径找到ImagePath，双击，就会弹出编辑字符串。我的原字符串为D:\Software\mysql-5.6\bin\mysqld MySQL，修改为"D:\Software\mysql-5.6\bin\mysqld" --defaults-file="D:\Software\mysql-5.6\my-default.ini" MySQL;
② 利用net stop mysql、net start mysql重启MySQL服务，然后重新使用cmd登录MySQL，输入show variables like '%character%';，发现character_set_database对应值为utf8。

3 Mysql + QT5.5.1

3.1 QT5.5.1安装

QT5.5.1的安装包下载。

百度云链接：https://pan.baidu.com/s/1RmkcwAavLGxF_HeV3PdwgA
提取码哟：lla0

双击后一路安装即可，其中设置安装路径为D:\Software\Qt，安装结束后将QT安装路劲下的D:\Software\Qt\Qt5.5.1\5.5\mingw492_32\bin加入环境变量，以自己的实际路劲为准。

注意，安装qt后编译程序时出现了Error while building/deploying project untitled错误提示，此时需要选择菜单栏 “工具”–》“选项”–》“构建和运行”， 在“概要”选项卡下面找到”Default build directory” 点击后面的“reset”按钮. 之后关闭QT，再重新打开就好了。
