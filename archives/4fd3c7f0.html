本节下载的文件也可从百度云中下载。

1.1 Win10下安装MySQL5.7.20

1.官网下载mysql-installer-community-5.7.20.0.msi，然后双击运行；

2.选择接受许可条款“I accept the license terms”,点击 “Next “，然后选择 “Server only”（表示只需要mysql服务器），点击 “Next “ 后如下图，可以选择安装路径及数据存储路径，这里默认不变。

继续点击next，可能出现如下情况，则需要先装所提示软件，点击退出安装。

从微软官网下载Visual Studio 2013 Redistributable，将vcredist_x64和vcredist_86都下载下来，先安装vcredist_x64后尝试重新安装mysql 5.7.20 ，假如依然报错，然后安装vcredist_x86后再次安装mysql 问题解决。

3.进入如下界面后，点击 “Execute” –> “next”；

4.一路 “next”，到下图界面，图中3306为默认端口号。

5.继续 “next”，设置root密码和服务器名。

6.然后一路”Execute” –> “next”，直到点击”Finish”。

7.安装完后，需要配置Mysql环境变量
Windows+R打开运行，输入sysdm.cpl，回车后如下图操作

8.以管理员的身份打开 cmd ，输入net start mysql57，(mysql57 是我的数据库名字，刚刚有改动的，请换成自己的数据库名字），如下图则成功；备注：关闭 mysql 的命令是net stop mysql57。

1.2 安装Navicat for MySQL

Navicat for MySQL 是管理和开发 MySQL 或 MariaDB 的理想解决方案，为数据库管理、开发和维护提供了一款直观而强大的图形界面。

1.下载前面百度云中的文件，双击安装 navicat110_mysql_cs_x64.exe，一路安装完成就好。

2.根据 readme.txt 中的提示，进行破解。

3.测试MySQL是否成功。

如果点击确认，可以可视化的看到数据库的具体情况，很直观。
